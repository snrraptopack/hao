# Changelog

## 0.3.0

- Rebuilt Auwla around setup-once components that return render closures.
- Removed public refs, watchers, stores, router, resource cache, plugins, and DSL APIs.
- Added event-driven rerendering for plain JSX event handlers.
- Added DOM patching with tag/key reuse, prop patching, event listener patching, and keyed child movement.
